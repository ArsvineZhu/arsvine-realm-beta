# Music Files

Place your audio files (`.mp3`, `.m4a`, `.flac`, `.wav`, `.ogg` — anything the
browser's native HTML5 `<audio>` can decode) here for the music player.

The playlist is configured in [`data/music.ts`](../../data/music.ts) — add or
remove tracks there, no component changes required.

> Audio files are excluded from git via `.gitignore`.
